Les différents blocs élémentaires

Les paragraphes

Un paragraphe est un bloc indispensable à la réalisation de pages web, en effet, les dernières normes xHTML interdisent de mettre du texte
directement dans le "body" des documents web, et recommandent d'utiliser l'élément p par défaut.

Les textes preformatés

public class ProgBonjour {
public static void main(String[] args) {
System.out.println("Bonjour le monde !");
}
}

Les listes

    Les listes ordonnées et non ordonnées permettent de faire toutes sortes d'énumérations :

  • premier point décomposée en sous-points:
    • ss-pt 1
    • ss-pt 2
  • deuxième point:
    • ss-pt
    • ss-pt
  • dernier point.

Et maintenant quelques définitions :

Document :
Écrit ou enregistrement électronique associé à une activité productrice ou consommatrice d'informations.
HTML :
Hypertexte Markup Language
WWW :
Wide-area hypermedia information retrieval initiative aiming to give universal access to a large universe of documents
URL :
Uniform Ressource Locator
